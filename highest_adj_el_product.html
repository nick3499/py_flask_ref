highest_adj_el_product.py

An array of positive and negative numbers is given:

arr = [3, 6, -2, -5, 7, 3]

The product of each pair of adjacent numbers is calculated:

product = arr[i] * arr[i+1]
  • 3 x 6
  • 6 x -2
  • -2 x -5
  • -5 x 7
  • 7 x 3 (highest product is 21) 
      
from flask import Flask, render_template

app = Flask(__name__)

@app.route('/')
def high_adj_num_prod():
    arr = [3, 6, -2, -5, 7, 3]
    product = arr[0] * arr[1]
    for i in range(len(arr)-1):
        if (arr[i] * arr[i+1]) > product:
            product = arr[i] * arr[i+1]
    return render_template("highest_adj_el_product.html", highest=product)

if __name__ == '__main__':
    app.run(debug=True)

highest_adj_el_product.sh

To run the Flask application, run the shell script in a Unix-like terminal emulator:

$ sudo ./highest_adj_el_product.sh

Setting development mode through a shell script avoids unexpected behavior. The Flask application is also included.

      
#!/bin/bash
# $ sudo ./highest_adj_el_product.sh

export FLASK_APP=highest_adj_el_product.py
export FLASK_ENV=development
flask run
